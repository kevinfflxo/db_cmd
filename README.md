## 查看版本

* **在資料庫未登入的時候**

```sql=
mysql -V
```

* **在資料庫登入的時候**

```sql=
SELECT VERSION();
```

## 在 MySQL console 執行 system 的指令

* **前面加個 system 即可**

```sql=
system cls
```
